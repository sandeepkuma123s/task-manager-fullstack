# 🚀 TaskFlow - Full Stack Task Manager

A modern Full Stack Task Management Application built using React, Node.js, Express.js, MongoDB Atlas, and JWT Authentication.

## 🌐 Live Demo

### Frontend

https://task-manager-fullstack-amber.vercel.app

### Backend API

https://task-manager-fullstack-t4ou.onrender.com

---

## 📌 Features

### Authentication

* User Registration
* User Login
* JWT Authentication
* Secure Protected Routes
* Logout Functionality

### Task Management

* Create Tasks
* View Tasks
* Update Tasks
* Delete Tasks
* Task Completion Tracking
* Priority Management (High, Medium, Low)

### Dashboard

* User Profile Section
* Search Tasks
* Filter Tasks
* Statistics Cards
* Calendar Integration
* Weekly Overview

### Database

* MongoDB Atlas Integration
* User-specific Task Storage
* Persistent Data After Refresh

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/sandeepkuma123s/task-manager-fullstack.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```

### Environment Variables

Create a `.env` file inside the server folder.

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
